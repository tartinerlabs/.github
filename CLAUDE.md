## Repository Purpose

This is a GitHub organization `.github` repository for **Tartiner Labs**. It contains organization-level GitHub configurations that apply across all repositories in the organization.

- `profile/README.md` - Organization profile displayed on https://github.com/tartinerlabs
- `README.md` - Repository description

## Common GitHub Organization Configs

This repository can be extended to include:
- `.github/ISSUE_TEMPLATE/` - Issue templates for all org repos
- `.github/PULL_REQUEST_TEMPLATE.md` - PR template for all org repos
- `.github/FUNDING.yml` - Sponsorship configuration
- `.github/workflows/` - Reusable workflow templates
